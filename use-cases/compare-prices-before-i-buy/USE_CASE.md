### 1. Title

Compare prices before I buy

### 2. Example prompt

"I want to buy this phone. Compare prices across stores and tell me when it drops below ₹35,000."

### 3. What the agent does

Searches trusted e-commerce stores for the product, pulls current prices and availability, and builds a side-by-side comparison with direct links. It then sets up a price watch and notifies you the moment the price drops below your target — so you never overpay or miss a deal.

### 4. Skills & tools used

- Web Search tool — queries stores like Amazon, Flipkart, Croma, and others for live pricing and availability
- Price Watcher skill — tracks the product price over time and triggers alerts when the threshold is crossed
- Notification tool — sends a price-drop alert via the user's preferred channel (email, Telegram, etc.)
