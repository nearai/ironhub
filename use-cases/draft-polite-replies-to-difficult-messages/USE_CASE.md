### 1. Title

Draft polite replies to difficult messages

### 2. Example prompt

"Help me reply politely to this angry customer / landlord / friend message."

### 3. What the agent does

Reads the incoming message and identifies the tone, intent, and underlying concern. It drafts a calm, constructive reply that acknowledges the other person's frustration without being defensive or dismissive. The agent produces 2–3 reply variations — a short and firm version, a warm and empathetic version, and a professional middle-ground — so you can pick the one that fits the situation best. It can also adjust tone on request (more apologetic, more assertive, more casual).

### 4. Skills & tools used

- Message Reply Writer skill — reads the original message and generates tone-appropriate responses with multiple variations
- Tone Adjuster skill — fine-tunes drafts to be more polite, firm, friendly, or professional based on the relationship and context
